   ```bash

2. В корень проекта скопируйте приложенные с ссылкой на гит .env и google_credits.json
3. В терминале 
    ```bash
   sudo docker compose up --build 
4. Приложение поднимется на http://127.0.0.1:8000, Swagger на http://127.0.0.1:8000/docs#
